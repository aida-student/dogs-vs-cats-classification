# dogs-vs-cats-classification
## Описание проекта Учебный проект по анализу изображений, посвящённый задаче классификации кошек и собак с использованием готовых библиотек машинного обучения.

## Цель работы
Освоить использование готовых библиотек машинного обучения
на примере задачи распознавания изображений.

## Используемые технологии
- Python
- TensorFlow
- Keras Applications

## Набор данных
Dogs vs Cats (Kaggle)
